# snen.dev


## Tech

Based on https://codesandbox.io/s/kind-sammet-j56ro (React 18 RC demo) but
using Deno 1.18.1.
A similar project (a framework for this sort of thing), which was a useful
reference for the site, can be found at https://ultrajs.dev/.

### How to read this code

Most directories have READMEs that should help describe how each file is used.
These should serve as a decent entrypoint.






## Design

The site design was built out using [figma](https://www.figma.com) and
[palettte](https://palettte.app/).
